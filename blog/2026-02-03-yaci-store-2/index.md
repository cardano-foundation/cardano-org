Yaci Store 2.0.0 is a high-performance Java-based indexer that introduces native ledger and governance state derivation, allowing applications to track proposals and calculate staking rewards directly from on-chain data. This update features a modular plugin framework supporting JavaScript and Python for custom processing. Operational enhancements like PostgreSQL range partitioning and Prometheus monitoring ensure the platform is scalable and production-ready for complex ada-based decentralized applications.
