The February 03, 2026, blog post by Satya Ranjan announces the release of Yaci Store 2.0.0, a high-performance Java-based indexer for the Cardano blockchain. This update introduces native ledger and governance state derivation, enabling applications to track proposals and calculate rewards directly. It also features a new plugin framework for custom data processing and operational enhancements like PostgreSQL partitioning and Prometheus monitoring, supporting scalable, production-ready deployments.
