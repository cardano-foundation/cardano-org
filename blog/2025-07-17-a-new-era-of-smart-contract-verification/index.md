A new tool has been introduced for the automated formal verification of smart contracts. By leveraging the Lean4 proof system and SMT solvers, the tool automatically checks that contracts behave as intended, eliminating the need for developers to write manual proofs. This innovation aims to make high-assurance development faster and more accessible by allowing for continuous verification within CI/CD pipelines.
