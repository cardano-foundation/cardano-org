---
title: DApp
slug: dapp
short: An application whose core logic runs as on-chain code rather than on a single company's servers, so its rules keep working even if any one operator walks away.
category: smart-contracts
level: beginner
link: /apps
aliases: [Decentralized Application]
mentalModel: "A regular app whose rules are enforced by on-chain code instead of a company's terms of service. The frontend can disappear; the rules cannot."
related: [smart-contract, wallet]
---

Most DApps still have a regular web frontend; what makes them "decentralized" is that the rules they enforce (swaps, lending, voting, etc.) live in smart contracts on the blockchain.
